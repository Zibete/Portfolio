# ✨ feat(ui): pulir estados interactivos visibles del portfolio

Base: `master`
Rama: `feat/011-interactive-polish-system`

## Qué se cambió
- `specs/features/011-interactive-polish-system/*`
  - se documentó alcance, plan, tareas y verificación de la feature
- `src/components/sections/stack-marquee.tsx` y `src/app/globals.css`
  - se incorporaron íconos por skill y acentos de color sobrios en el marquee
  - se reemplazó el loop CSS por un desplazamiento controlado para mantener hover estable y desaceleración progresiva
  - se dejó `prefers-reduced-motion` sin movimiento continuo en el marquee
- `src/components/layout/site-header.tsx` y `src/app/globals.css`
  - se mejoraron `hover`, `active` y `focus-visible` del header con `nav-pill`
  - la ruta activa ahora se resuelve con `usePathname()`
- `src/app/globals.css`
  - se agregaron `interactive-card`, `interactive-card-subtle` e `interactive-chip`
  - se unificó feedback visual sutil para hover en cards, chips y nav sin abrir un sistema nuevo de layout
- `src/app/page.tsx`, `src/components/sections/home-hero.tsx` y `src/components/sections/featured-project.tsx`
  - se aplicó polish interactivo visible en Home y en el proyecto destacado
- `src/app/projects/page.tsx` y `src/app/projects/[slug]/page.tsx`
  - se mejoró el feedback visual de project cards, chips y preview wrappers
- `src/components/shared/page-section.tsx`, `src/components/shared/page-aside.tsx`, `src/app/about/page.tsx`, `src/app/cv/page.tsx`, `src/app/experience/page.tsx`, `src/app/contact/page.tsx` y `src/app/skills/page.tsx`
  - se sumó iluminación sutil en surfaces/cards visibles para que las internas no queden completamente muertas al hover

## Por qué
- el sitio ya tenía base visual sólida, pero faltaba feedback interactivo consistente en navegación, marquee y superficies clave
- el marquee necesitaba un hover estable y predecible para que el destaque por skill no se sintiera roto
- el alcance final se recortó para sostener una mejora visible y defendible, sin prometer un sistema global de elevación de cards
