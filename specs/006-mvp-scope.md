# 006 Alcance del MVP

## Dentro de alcance para la iteración fundacional actual

- inicialización del proyecto Next.js
- configuración de TypeScript, ESLint, Tailwind y App Router
- instalación y preparación de shadcn/ui, MDX y el scaffold de Supabase
- creación de la estructura base de carpetas
- creación de documentos fundacionales y specs
- dejar un app shell mínimo y navegable

## Dentro de alcance para el futuro MVP

- contenido real de Sobre mí
- proyectos curados iniciales
- resúmenes de experiencia
- definición de habilidades
- CV descargable
- flujo mínimo de contacto

## Fuera de alcance de forma explícita

- una pasada completa de diseño del portfolio
- autenticación
- dashboard admin
- blog o notas
- backend avanzado
- analytics y tracking
- scraping automático de contenido desde GitHub

## Criterios de aceptación de esta iteración

- la app compila correctamente
- lint pasa
- existen rutas para todas las secciones planificadas
- docs y specs están presentes y son útiles
- el repo queda listo para una segunda iteración centrada en contenido
