# 003 Sistema de diseño

## Dirección visual

- dark-first
- sensación fuerte y cercana a un producto tech
- superficies limpias con contraste controlado
- sin una paleta dominada por violeta

## Tono de la interfaz

- profesional
- directo
- moderno
- conciso

## Dirección tipográfica

- una tipografía display expresiva para hero y títulos de sección
- una sans legible para texto corrido
- la jerarquía debe resolver la mayor parte del trabajo visual

## Base de UI

- Tailwind CSS para layout y tokens
- shadcn/ui como baseline de componentes
- superficie mínima de componentes en la fase fundacional

## Set inicial de componentes

- button
- header
- footer
- secciones de contenido temporal
- theme toggle

## Reglas de interacción

- el tema dark/light debe funcionar desde el inicio
- la motion debe mantenerse sutil hasta diseñar la landing real
- estados de foco y legibilidad importan más que el adorno
