# Verification — Feature 003 Hero identity

## Validación automática

Ejecutar:
```bash
npm run lint
npm run build
```

## Validación manual

Levantar:
```bash
npm run dev
```

### Hero
- [ ] El nombre principal tiene más presencia
- [ ] La entrada/animación se siente sutil y moderna
- [ ] El badge en español se integra de forma natural
- [ ] Las CTAs tienen mejor jerarquía

### Home
- [ ] El hero convive bien con el resto de la composición
- [ ] No se siente recargado
- [ ] Sigue alineado con la estética producto tech

### Dark mode
- [ ] El hero se ve fuerte y premium
- [ ] La animación no rompe contraste ni lectura

### Light mode
- [ ] El hero sigue siendo limpio
- [ ] El badge y la tipografía conservan legibilidad
- [ ] No aparecen nuevas interferencias visuales

### Técnica
- [ ] No hay errores de hidratación
- [ ] No hay saltos bruscos de layout
- [ ] No hay imports rotos
- [ ] No hay dependencias innecesarias agregadas

## Criterio de done

La feature se considera terminada cuando:

- [ ] El hero ya tiene una identidad más fuerte y memorable
- [ ] El nombre y el badge aportan valor real a la primera impresión
- [ ] La animación se siente sutil y mantenible
- [ ] Lint y build pasan
- [ ] El cambio no invade el alcance de futuras features visuales
