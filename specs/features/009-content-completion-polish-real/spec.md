# Feature 009 — Content completion polish real

## Objetivo

Reemplazar el contenido temporal del portfolio por contenido real, profesional y publicable, manteniendo el sistema visual actual y sin abrir alcance funcional nuevo.

## Alcance

- reemplazar copy meta por narrativa profesional concreta
- crear las primeras entradas reales de experiencia si aportan claridad
- sumar el segundo proyecto público relevante si mejora la coherencia del portfolio
- ajustar copy complementario en `site-config.ts` y `README.md` si queda desalineado

### No entra en esta feature

- agregar backend de contacto
- publicar PDF real del CV

- agregar dependencias nuevas

## Resultado esperado

Al finalizar, el portfolio debe hablar de Matías Peralta y de su valor profesional con un tono claro, escaneable y defendible, sin páginas que todavía se lean como placeholder.
