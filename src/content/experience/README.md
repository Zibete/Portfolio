# Contenido de experiencia

Guardar un archivo `.mdx` por entrada de experiencia en este directorio.

- Usar un frontmatter conciso y enfocado en impacto de negocio y alcance tecnico.
- Mantener el modelo de contenido alineado con `specs/005-content-model.md`.
- Definir `order` cuando el orden visual de las entradas importe.
- Priorizar resumenes breves y escaneables por sobre cronologias largas.
