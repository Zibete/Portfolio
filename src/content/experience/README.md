# Contenido de experiencia

Guardá un archivo `.mdx` por entrada de experiencia en este directorio.

- Usá un frontmatter conciso y enfocado en impacto de negocio y alcance técnico.
- Mantené el modelo de contenido alineado con `specs/005-content-model.md`.
- Priorizá resúmenes breves y escaneables por sobre cronologías largas.
